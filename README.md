# openCV-Basics
Ass1 - A beginner-level OpenCV project in Python that loads an image from a user-provided path and uses a menu-driven loop to convert the image to grayscale, display it, or save it. The program demonstrates basic OpenCV functions, loops, conditionals, and user input handling.
Ass2 - A beginner OpenCV project in Python that loads an image and provides a menu-driven interface to add text, draw lines, rectangles, and circles on the image. The project demonstrates OpenCV drawing functions, user input handling, loops, and conditional logic.
saving_video - A beginner OpenCV project in Python that captures live video from a webcam, displays it in real time, and records the footage to an MP4 file using VideoCapture and VideoWriter. The project demonstrates basic video processing, frame handling, and proper resource management.
